# Necesse - QoL Chests  

A basic modpack for quality of life chests. Introduces a suite of new chests with larger storage sizes, based on materials used in the game. 
Based from the 'example mod' of Necesse.

## Chests available

Currently supports a 50 and 60 slot chest for demonic and tungsten ingots.

## Assets

Assets are currently color skins of [SoulChasm](https://github.com/Azuraei/Soul-Chasm) chest assets. Final assets are
pending post testing.
